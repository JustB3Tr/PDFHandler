# PDFHandler

PDFHandler is a desktop app that converts PDFs into slideshow-friendly formats.

## Features
- Drag-and-drop PDF import
- Source preview panel
- Converted preview panel
- Export to PowerPoint (.pptx)
- Google Slides-compatible PPTX export
- PNG slide set export
- JPG slide set export
- 300 / 450 / 600 DPI output
